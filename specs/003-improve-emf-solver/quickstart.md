# Quickstart

1. Install dependencies:
   `npm install`
2. Start the app:
   `npm run dev`
3. Open the lab page and try the multi-emitter presets in `/lab`.
4. Toggle the FPS counter and switch LOD / solver profile settings to compare responsiveness.
5. Run checks before handing off:
   `npm test`
   `npm run lint`
   `npm run type-check`
